Common Pasture

A game about the Tragedy of the Commons

Based on the board game created by Jaroslaw Flis

Join game (student) Instructor panel

How to start a game?

  1. Open the Instructor panel on your computer
  2. A room code and QR code will appear on screen
  3. Display the dashboard on a projector or share your screen
  4. Students scan the QR code with their phones or type the address
  5. Students enter the room code and their name
  6. When everyone has joined, press Start
Everyone must be connected to the same WiFi network
